## ✨ Features


- 🎟️ Interactive Cinema Seat Booking System
- 🪑 Real-time Seat Selection & Deselection
- 🚫 Booked Seats Disabled Automatically
- 💎 Multiple Seat Categories
  - Regular Seats
  - Premium Seats
  - VIP Seats
- 🎨 Dynamic Seat Colors Based on Category
- 💰 Automatic Total Price Calculation
- 📱 Fully Responsive Design
- ⚡ Smooth Hover & Selection Animations
- 🧾 Booking Summary Section
- 🔒 Prevents Booking Without Seat Selection
- 🏷️ Seat Labels with Row & Seat Numbers
- 🎬 Cinema Screen Layout UI
- 🛣️ Center Aisle Support
- 🔄 Live State Management using React Hooks
- 🧠 Built with TypeScript Support
- 🎯 Clean UI using Tailwind CSS
- ⚛️ Developed with Next.js & React
